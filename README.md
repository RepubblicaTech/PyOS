# PyOS
A simple Python OS. 99,9% using pure Python (almost no `os.system()` stuff)

## Commands

### `help`

Displays available commands.

### `exit`

exit.

### `ver`

Displays system version

### `prsh`

The Print Shell ported from the "Legacy" to vNext
